# lift-ng-workshop

To start container just run:
```bash
./sbt
> container:start
```

To reload after class changes inside sbt run:
```bash
> ~; compile; container:reload /
```

Project is based on [lift_26_sbt](https://github.com/lift/lift_26_sbt/tree/master/scala_211/lift_basic)
